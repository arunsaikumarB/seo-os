# SEO OS — How to Use (Backlink Builder)

**Who this is for:** Anyone using SEO OS Version 1.0 as a backlink product.  
**What you will learn:** How to go from login → import websites → win and verify backlinks.

---

## What SEO OS is now

SEO OS is an **AI Backlink Builder**.

You work in one main product flow:

```
Import websites
      ↓
AI Analysis (scores opportunities)
      ↓
Opportunity Queue → Pipeline → Campaigns
      ↓
Outreach
      ↓
Link Verification → Reports
```

Other engines (Browser Intelligence, Knowledge, Memory, Relationships, Campaigns) still run **in the background** when you import or analyze. You do not need to open those modules separately.

---

## Before you start

1. **Sign up / Log in**
2. **Create an Organization** (your company or agency)
3. **Create a Project** (one project = one website you want backlinks for)
4. Open the project → you land near **Mission Control**

Keep using the correct project. Switching projects switches all data.

## Step-by-step: win your first backlinks

### Step 1 — Open Mission Control
Check these numbers (they update from live data):

- Imported Websites  
- Qualified Opportunities  
- Outreach Queue  
- Emails Sent  
- Replies  
- Backlinks Won  
- Verified Links  
- Success Rate  

If everything is `0`, you have not imported yet — that is normal.

**Click:** Import websites (or open **Backlink Builder → Import Websites**).

---

### Step 2 — Import Websites
1. Open **Backlink Builder → Import Websites**
2. Paste URLs (one per line), for example:

```
https://example-blog.com
https://industry-directory.org
https://niche-forum.com
```

3. Import / validate the list  
4. Run the automation / analysis when prompted  

**What happens automatically in the background:**
- Domains are analyzed and scored into opportunities  
- Memory + Knowledge notes can be written for the AI  
- Relationship records are seeded  
- Light Browser Intelligence scans may start  

You stay inside Backlink Builder — no need to open other modules.

---

### Step 3 — AI Analysis
1. Open **Backlink Builder → AI Analysis**
2. Confirm the run finished  
3. Opportunities should appear as **qualified / prepared** for review  

Tip: Higher scores = better priority for outreach.

---

### Step 4 — Review Opportunity Queue
1. Open **Opportunity Queue**
2. Read titles, domains, and scores  
3. Approve the ones worth pursuing  
4. Reject weak or irrelevant sites  

Only approved opportunities should move into campaigns.

---

### Step 5 — Pipeline & Campaigns
1. Open **Pipeline** to see stages (discovered → qualified → outreach → won / verified)  
2. Open **Campaigns**  
3. Create or open a campaign  
4. Attach approved opportunities  

A campaign keeps outreach organized instead of one-off emails.

---

### Step 6 — Outreach
1. Open **Outreach**  
2. Draft or generate emails (use **SEO AI Assistant** if you need copy help)  
3. Review before sending  
4. Send / follow up from the inbox  

Connect email under **Org → Integrations** if sending is not configured yet.

---

### Step 7 — Relationships (optional but useful)
Open **Relationships** to see companies/contacts created from imports and outreach.  
Use this when you need context before a follow-up.

---

### Step 8 — Link Verification
1. Open **Link Verification**  
2. Check claimed or pending links  
3. Mark verified when the live backlink exists  
4. Wins appear in Mission Control as **Backlinks Won / Verified Links**

---

### Step 9 — Reports
1. Open **Reports**  
2. Generate a backlink / campaign / monthly report  
3. Export for your client or team

## SEO AI Assistant (how to use chat)

Open **SEO AI Assistant** from Home.

Useful prompts:

- Find high-value backlink opportunities  
- Prioritize my opportunity queue  
- Draft a guest post outreach email  
- Which prospects should I contact first?  
- Summarize imported websites  
- Build an outreach sequence  
- Check link verification status  

Use it for drafting and decisions — still review outreach before sending.

## Common questions

**Where did Knowledge / Memory / Browser Intelligence go?**  
They still run after import/analysis. They are hidden from the sidebar so Version 1.0 stays focused on backlinks.

**Why are my KPIs zero?**  
Import websites and run AI Analysis first.

**Can I work on multiple websites?**  
Yes — create one Project per website.

**Do I need Expert mode?**  
No. Version 1.0 uses the Backlink Builder navigation for everyone.
